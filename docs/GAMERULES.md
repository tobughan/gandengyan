## 三、核心游戏流程

### 3.1 开局流程
```
1. 选择玩家人数（3/4/5人）
2. 选择AI难度
3. 4人摇骰子定庄家
4. 发牌：庄家6张，闲家各5张
5. 庄家先出牌
```

### 3.2 出牌回合流程
```
1. 当前玩家出牌
2. 下家按逆时针顺序尝试接牌
3. 能接 → 接牌后成为当前出牌者，继续下家接牌
4. 不能接 → 过牌，轮到下一家
5. 所有人都过牌 → 一轮结束
```

### 3.3 一轮结束后处理
```
1. 检查最后出牌者手牌是否为空
   → 为空 → 获胜，游戏结束
   → 不为空 → 继续
2. 检查底牌是否还有剩余
   → 底牌耗尽 → 进入硬打阶段，不补牌
   → 有底牌 → 所有人依次摸1张
3. 从最后出牌者开始，按顺序每人摸1张
4. 最后出牌者继续出牌
```

## 四、接牌规则

### 4.1 通用规则
- 必须使用 **相同牌型** 接牌
- 必须 **恰好大1点** 接牌
- 相同长度牌型才能接（顺子/连对长度必须一致）

### 4.2 2的特殊功能
| 场景 | 规则 |
|------|------|
| 单2 | 可以接任意单张牌 |
| 对2 | 可以接任意对子 |
| 单2限制 | 不可作为最后一手打出 |
| 强制出牌 | 手牌剩2张且含2时，必须先出2 |

### 4.3 接牌示例
| 出牌 | 可接 | 不可接 |
|------|------|--------|
| 3 | 4、2 | 5、6... |
| 33 | 44、22 | 55、66... |
| 345 | 456 | 567、234 |
| 3344 | 4455 | 5566 |

## 五、炸弹层级系统

### 5.1 炸数对照表
| 牌型 | 炸数 | 说明 |
|------|------|------|
| 3张相同 | 1炸 | 基础炸弹 |
| 4张相同 | 2炸 | |
| 王炸 | 2炸 | 所有2炸中最大 |
| 5张相同 | 3炸 | |
| 6张相同 | 4炸 | |
| 7张相同 | 5炸 | 理论最大 |

### 5.2 炸弹比较规则
1. **先比炸数**: 炸数高的 > 炸数低的
2. **炸数相同**:
   - 王炸 > 所有其他2炸
   - 普通炸弹按点数比较

### 5.3 炸弹优先级表（从高到低）
```
7张相同 > 6张相同 > 5张相同 > 王炸 > 4张相同 > 3张相同
```

### 5.4 炸弹使用规则
- 炸弹可以炸任何非炸弹牌型
- 炸弹之间按上述优先级互炸

## 六、计分系统

### 6.1 基础计分公式
```
基础分 = 剩余牌数 × (1 + 总炸数)
```

### 6.2 关状态特殊计分
| 玩家类型 | 触发条件 | 计分公式 |
|----------|----------|----------|
| 庄家 | 整局只出了1次牌 | 20 + 10 × 总炸数 |
| 闲家 | 整局一张牌都没出 | 20 + 10 × 总炸数 |

### 6.3 总炸数计算
- 本局游戏中所有玩家打出的炸弹炸数之和
- 王炸计2炸

### 6.4 庄家轮换
- 上一局赢家自动成为下一局庄家

## 七、特殊场景处理

| 场景 | 处理规则 |
|------|----------|
| 玩家只剩1张2 | 不能打出，必须补牌后组合出 |
| 玩家剩2张牌含2 | 必须先出2，不能留2到最后 |
| 底牌不足所有人摸 | 按顺序摸，摸完即止，剩余人不摸 |
| 万能牌配顺子 | 可替代中间缺失的点数（如34+王=345） |
| 万能牌配连对 | 可替代缺失的对子（如33+王+王=3344） |

## 八、AI难度设计

| 难度 | 出牌策略 |
|------|----------|
| **简单** | 随机选择合法出牌，不考虑最优策略 |
| **普通** | 小牌先出，保留2和炸弹，基础配牌策略 |
| **困难** | 算牌+记牌，万能牌优先配炸弹，博弈策略 |
